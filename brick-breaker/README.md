# Brick Breaker

A modern web-based Brick Breaker (Arkanoid) game.

## Features
- Power-ups: expand paddle, multi-ball, extra life, slow ball
- Progressive levels and scoring
- Touch and keyboard controls
- Responsive design

## Controls
- Arrow keys or mouse/touch to move paddle
- Space or tap to launch ball

## How to Play
1. Break all bricks to advance to the next level.
2. Catch power-ups for bonuses.
