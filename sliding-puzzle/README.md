# Sliding Puzzle (15-puzzle)

A classic sliding tile puzzle. Arrange the tiles in order by sliding them into the empty space!

## Features
- Simple, responsive UI
- 4x4 grid (15 tiles)
- Move counter
- Restart button
- No libraries required
